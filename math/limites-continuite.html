Chapitre 1

Asymptotes : limites en l'infini et en un point

Une asymptote est une droite dont la courbe se rapproche infiniment. Horizontale quand x → ±∞ (la limite est un nombre), verticale quand x s'approche d'une valeur interdite (la limite est ±∞). L'exemple ci-dessous a les deux.

f(x) = 2x + 1x − 1 = 2 + 3x − 1   →  asymptote horizontale y = 2, asymptote verticale x = 1
f(x₀)
—
écart |f(x₀) − 2|
—

Pousse x₀ vers la droite : f(x₀) se colle à 2 (l'écart fond vers 0) — c'est l'asymptote horizontale. À l'inverse, approche x₀ de 1 : f explose vers l'infini — c'est l'asymptote verticale. On note limx→+∞ f(x) = 2 et limx→1⁺ f(x) = +∞.

Chapitre 2

Opérations & formes indéterminées

Souvent on combine des limites connues. Mais quatre situations ne se décident pas directement : ce sont les formes indéterminées (FI). La parade la plus fréquente : factoriser par le terme dominant.

Les 4 FI :
∞ − ∞   0 × ∞
∞∞    00

À l'infini, un polynôme a la même limite que son terme de plus haut degré. Un quotient de polynômes a la même limite que le quotient des termes dominants.

…

P(x) = a·x³ − 2x² + 1. Quand x → +∞, c'est le terme a·x³ qui « gagne » : la limite ne dépend que de son signe. Change a et regarde la branche de droite basculer.

Chapitre 3

La continuité : tracer sans lever le crayon

Une fonction est continue sur un intervalle si on peut tracer sa courbe d'un seul trait, sans saut. Formellement : limx→a f(x) = f(a) en tout point a. Règle la hauteur du saut et observe.

…
Continue — la courbe se referme : pas de saut, on trace d'un trait.
Point fermé ● — la valeur f(a) prise par la fonction.
Point ouvert ○ — la valeur « approchée » mais non atteinte.
À retenir — toute fonction dérivable est continue (la réciproque est fausse).

Au Bac, on admet que les fonctions usuelles (polynômes, exp, ln, racine, fonctions rationnelles sur leur domaine…) sont continues. La continuité est l'hypothèse indispensable du théorème qui suit.

Chapitre 4

Le théorème des valeurs intermédiaires

Si f est continue sur [a, b], alors elle prend toutes les valeurs comprises entre f(a) et f(b). Si en plus f est strictement monotone, cette valeur est atteinte une seule fois.

f continue sur [a,b]  et  k entre f(a) et f(b)  ⇒  ∃ c ∈ [a,b], f(c) = k

—
f(2)
—
solution c
—

Ici f(x) = x³ + x est continue et strictement croissante : pour chaque hauteur k, la droite horizontale coupe la courbe en un unique point c. C'est ce qui justifie qu'une équation f(x) = k admet une solution unique — sans pour autant savoir la calculer.

Chapitre 5

La dichotomie : encadrer la solution

Le TVI dit qu'une solution existe, mais pas sa valeur. Pour l'approcher, on coupe l'intervalle en deux à chaque étape et on garde la moitié qui contient la solution. La précision double à chaque coup.

Résoudre x³ + x = 5 sur [1 ; 2]  (solution ≈ 1,5156…)

0
intervalle [a ; b]
[1 ; 2]

À chaque étape, on calcule le signe de f au milieu m : s'il est de même signe que f(a), la solution est à droite (a ← m), sinon à gauche (b ← m). Après n coupes, la solution est connue à (b−a)/2ⁿ près — c'est l'algorithme qu'on programme en Python au Bac.

Cherche, puis déplie la correction.

★ ☆ ☆ — limite en l'infini

1. Limite d'un quotient

Détermine limx→+∞ de f(x) = (3x² − x + 2)/(x² + 5).
Voir la correction

On factorise haut et bas par le terme dominant x² :

f(x) = x²(3 − 1/x + 2/x²) / [x²(1 + 5/x²)] = (3 − 1/x + 2/x²) / (1 + 5/x²)

Quand x → +∞ : 1/x → 0, 2/x² → 0, 5/x² → 0.

limx→+∞ f(x) = 3/1 = 3 (asymptote horizontale y = 3).
★ ★ ☆ — forme indéterminée

2. Lever une indétermination « ∞ − ∞ »

Calcule limx→+∞ (√(x² + x) − x).
Voir la correction

Forme « ∞ − ∞ ». On multiplie par la quantité conjuguée :

√(x²+x) − x = [(√(x²+x) − x)(√(x²+x) + x)] / (√(x²+x) + x) = (x² + x − x²) / (√(x²+x) + x) = x / (√(x²+x) + x)

On factorise par x (avec x > 0, √(x²+x) = x·√(1 + 1/x)) :

= x / [x(√(1+1/x) + 1)] = 1 / (√(1+1/x) + 1) → 1/(1+1)
limx→+∞ (√(x²+x) − x) = 1/2.
★ ★ ☆ — TVI

3. Existence et unicité d'une solution

Soit f(x) = x³ + 3x − 1 sur ℝ. Montre que l'équation f(x) = 0 admet une unique solution α, puis donne un encadrement de α à 0,1 près.
Voir la correction

f est un polynôme, donc continue sur ℝ. Sa dérivée f′(x) = 3x² + 3 > 0, donc f est strictement croissante sur ℝ.

f(0) = −1 < 0 et f(1) = 1 + 3 − 1 = 3 > 0 : 0 est compris entre f(0) et f(1).

Par le TVI (version stricte monotonie), f(x) = 0 admet une solution unique α, et α ∈ ]0 ; 1[.

On affine : f(0,3) = 0,027 + 0,9 − 1 = −0,073 < 0 ; f(0,4) = 0,064 + 1,2 − 1 = 0,264 > 0.

Donc 0,3 < α < 0,4 : encadrement à 0,1 près.
★ ★ ★ — dichotomie / Python

4. L'algorithme de dichotomie

On cherche la solution de x³ + x = 5 sur [1 ; 2] à 0,01 près. Combien d'étapes de dichotomie faut-il au minimum ? Écris l'idée de l'algorithme.
Voir la correction

Après n coupes, la largeur de l'intervalle est (2−1)/2ⁿ = 1/2ⁿ. On veut 1/2ⁿ ≤ 0,01, soit 2ⁿ ≥ 100.

2⁶ = 64 < 100, 2⁷ = 128 ≥ 100
Il faut au minimum 7 étapes.

Algorithme (principe) :

a, b = 1, 2 tant que b - a > 0.01 : m = (a + b)/2 si (m**3 + m - 5)*(a**3 + a - 5) <= 0 : b = m # la solution est dans [a, m] sinon : a = m # la solution est dans [m, b]
On renvoie m comme valeur approchée de la solution.
Pour appliquer le TVI : énonce continuité, monotonie, puis compare la valeur cible aux bornes. Les trois, toujours.
Avant de fermer le cahier

Les réflexes à retenir

  • Asymptote horizontale — lim en ±∞ finie = ℓ ⇒ droite y = ℓ.
  • Asymptote verticale — lim infinie en a ⇒ droite x = a.
  • Terme dominant — limite d'un polynôme/quotient en ±∞ = celle des termes de plus haut degré.
  • 4 formes indéterminées — ∞−∞, 0×∞, ∞/∞, 0/0. On factorise (ou conjugue).
  • Continue — tracé d'un trait ; limx→a f = f(a). Dérivable ⇒ continue.
  • TVI — continue + k entre f(a) et f(b) ⇒ solution existe ; + stricte monotonie ⇒ unique.
  • Dichotomie — on coupe en deux, précision (b−a)/2ⁿ.
