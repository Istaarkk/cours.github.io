La fonction
exponentielle

La seule fonction qui est égale à sa propre dérivée : plus elle est haute, plus elle monte vite. Définition, propriétés algébriques (somme → produit), courbe et limites, dérivée des composées eᵘ, croissances comparées et résolution d'équations. Tout illustré par des courbes à explorer.

Chapitre 1

La fonction égale à sa dérivée

L'exponentielle est définie comme l'unique fonction telle que f′ = f et f(0) = 1. Autrement dit, en chaque point, la pente de la tangente est égale à la hauteur de la courbe. Déplace le point et vérifie.

exp′ = exp  ·  exp(0) = 1   |   on note exp(x) = eˣ  avec  e ≈ 2,718

—
pente f′(a) = eᵃ
—
les deux sont…
égaux

La barre verticale cyan (la hauteur eᵃ) a la même valeur que la pente de la tangente orange. C'est cette propriété d'auto-renforcement qui explique la croissance « explosive » : plus c'est haut, plus ça grimpe vite.

Chapitre 2

Somme → produit : la relation fonctionnelle

La règle-clé, exactement l'inverse du logarithme : e^(a+b) = eᵃ × eᵇ. Ajouter dans l'exposant revient à multiplier les valeurs. Règle a et b et regarde les hauteurs se multiplier.

e^(a+b) = eᵃ × eᵇ  ·  e^(−a) = 1/eᵃ  ·  e^(a−b) = eᵃ/eᵇ  ·  (eᵃ)ⁿ = e^(na)

La hauteur e^(a+b) (3ᵉ barre) est exactement le produit des deux premières — pas leur somme. C'est ce qui fait de l'exponentielle l'outil naturel des phénomènes à croissance proportionnelle (population, intérêts composés, désintégration).

Chapitre 3

Courbe, variations et limites

eˣ est définie sur ℝ, strictement positive (toujours au-dessus de l'axe) et strictement croissante. Aux bords : elle s'écrase sur 0 vers −∞ et explose vers +∞.

eˣ > 0 pour tout x  ·  limx→−∞ eˣ = 0  (asymptote horizontale y = 0) |  limx→+∞ eˣ = +∞

Repères : e⁰ = 1 (la courbe coupe l'axe vertical en 1) et e¹ = e ≈ 2,72. Comme eˣ est strictement croissante, elle est bijective : eᵃ = eᵇ ⇔ a = b et eᵃ < eᵇ ⇔ a < b — d'où la résolution d'équations.

Chapitre 4

Dérivée composée & croissances comparées

La dérivée de eᵘ fait apparaître u′ : (eᵘ)′ = u′·eᵘ. Et à l'infini, l'exponentielle écrase toute puissance de x — c'est ce qui lève les indéterminations.

(eˣ)′ = eˣ  ·  (eᵘ)′ = u′·eᵘ   |   limx→+∞ eˣ/x = +∞  ·  limx→−∞ x·eˣ = 0

La courbe eˣ finit toujours par dépasser la parabole x² (et n'importe quelle puissance). C'est ce que traduit eˣ/x → +∞ : un grand classique du Bac pour lever les formes « ∞/∞ » mêlant exponentielle et polynômes.

Chapitre 5

Résoudre & le lien avec ln

exp et ln sont réciproques : l'une défait l'autre. C'est l'aller-retour qui résout la plupart des équations.

Les passages clés

eᵃ = eᵇ  ⇔  a = b

eˣ = k  ⇔  x = ln k (avec k > 0)

eᵃ < eᵇ  ⇔  a < b (exp croissante)

ln(eˣ) = x  ·  e^(ln x) = x (x > 0)

L'astuce du changement de variable

Pour une équation du type e^(2x) − 3eˣ + 2 = 0, on pose X = eˣ (avec X > 0) : on obtient une équation du second degré X² − 3X + 2 = 0, qu'on résout, puis on revient à x par x = ln X.

eˣ est toujours > 0 : une « solution » X = eˣ négative ou nulle est à rejeter avant de prendre le ln.

Exercices type Bac

Cherche, puis déplie la correction.

★ ☆ ☆ — simplification

1. Simplifier une expression

Simplifie A = e⁵ × e^(−3) / e².
Voir la correction

On additionne et soustrait les exposants :

A = e^(5 − 3 − 2) = e⁰
A = 1.
★ ★ ☆ — équation

2. Équation avec changement de variable

Résous dans ℝ : e^(2x) − 3eˣ + 2 = 0.
Voir la correction

On pose X = eˣ (donc X > 0). Comme e^(2x) = (eˣ)² = X² :

X² − 3X + 2 = 0 → (X − 1)(X − 2) = 0 X = 1 ou X = 2 (toutes deux > 0, acceptées)

On revient à x : eˣ = 1 ⇒ x = 0 ; eˣ = 2 ⇒ x = ln 2.

S = {0 ; ln 2}.
★ ★ ☆ — inéquation

3. Résoudre une inéquation

Résous : e^(2x) ≤ e^(x + 3).
Voir la correction

exp est strictement croissante, donc on compare directement les exposants :

e^(2x) ≤ e^(x+3) ⇔ 2x ≤ x + 3 ⇔ x ≤ 3
S = ] −∞ ; 3 ].
★ ★ ★ — étude de fonction

4. Variations et limites de x·e⁻ˣ

Soit f(x) = x·e^(−x) sur ℝ.
  1. Calcule f′(x) et étudie son signe.
  2. Donne les limites en −∞ et +∞, et le maximum de f.
Voir la correction

1. Produit u·v avec u = x, v = e^(−x), v′ = −e^(−x) :

f′(x) = 1·e^(−x) + x·(−e^(−x)) = e^(−x)(1 − x)

Or e^(−x) > 0, donc f′ a le signe de (1 − x) : positif si x < 1, négatif si x > 1.

f croît sur ]−∞ ; 1], décroît sur [1 ; +∞[ ; maximum f(1) = 1·e^(−1) = 1/e ≈ 0,368.

2. En −∞ : x → −∞ et e^(−x) → +∞, le produit → −∞. En +∞ : par croissances comparées, x·e^(−x) = x/eˣ → 0.

lim−∞ f = −∞ ; lim+∞ f = 0 (asymptote y = 0).
Repère la forme eᵘ et n'oublie jamais le facteur u′ en dérivant. Pour les équations : pose X = eˣ avec X > 0.

Les réflexes à retenir

  • Définition — exp′ = exp et exp(0) = 1 : pente = hauteur en tout point.
  • Somme → produit — e^(a+b)=eᵃeᵇ ; e^(−a)=1/eᵃ ; (eᵃ)ⁿ=e^(na).
  • Signe & variations — eˣ > 0 toujours, strictement croissante sur ℝ.
  • Limites — lim−∞ eˣ = 0 (asymptote y=0) ; lim+∞ eˣ = +∞.
  • Dérivée — (eˣ)′ = eˣ ; (eᵘ)′ = u′·eᵘ.
  • Croissances comparées — eˣ/x → +∞ ; x·eˣ → 0 (x→−∞).
  • Équations — eᵃ=eᵇ ⇔ a=b ; eˣ=k ⇔ x=ln k. Changement X=eˣ > 0.
