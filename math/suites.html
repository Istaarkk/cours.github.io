Une suite est croissante si uₙ₊₁ ≥ uₙ pour tout n, décroissante si uₙ₊₁ ≤ uₙ. Pour le prouver, on étudie le signe de la différence uₙ₊₁ − uₙ, ou (si tous les termes sont positifs) on compare le rapport uₙ₊₁ / uₙ à 1.

Chapitre 2

Arithmétique ou géométrique ?

Deux modèles à reconnaître absolument. Arithmétique : on ajoute toujours la même raison r. Géométrique : on multiplie toujours par la même raison q. Bascule entre les deux et règle u₀ et la raison.

…

Arithmétique

Récurrence : uₙ₊₁ = uₙ + r  ·  Explicite : uₙ = u₀ + n·r

Croissante si r > 0, décroissante si r < 0. Les points sont alignés (croissance linéaire).

Géométrique

Récurrence : uₙ₊₁ = q·uₙ  ·  Explicite : uₙ = u₀·qⁿ

Croissance (ou décroissance) exponentielle. Le comportement à l'infini dépend entièrement de q (chapitre 4).

Chapitre 3

La limite : se rapprocher aussi près qu'on veut

Dire que uₙ tend vers ℓ, c'est dire : pour toute bande aussi fine qu'on veut autour de ℓ, tous les termes finissent par y entrer et y rester. Resserre la bande (ε) et regarde à partir de quel rang la suite y reste.

uₙ = 3n + 1n + 2  →  ℓ = 3   car  |uₙ − 3| = 5n + 2 → 0
limite ℓ
3

—
rang N (entrée définitive)

Aussi petite que soit la bande, il existe toujours un rang N au-delà duquel tous les points sont dedans. C'est ça, « converger vers ℓ ». Une suite qui ne se stabilise vers aucun nombre fini est dite divergente (elle peut tendre vers ±∞, ou osciller).

Chapitre 4

Le théorème-clé : la limite de qⁿ

Pour les suites géométriques, tout se joue sur la raison q. Fais glisser q et regarde le comportement de qⁿ basculer.

…
−1 < q < 1 → qⁿ tend vers 0.
q = 1 → suite constante égale à 1.
q > 1 → qⁿ tend vers +∞.
q ≤ −1 → qⁿ n'a pas de limite (les signes alternent).
Chapitre 5

Le raisonnement par récurrence

Pour prouver qu'une propriété P(n) est vraie pour tout n, on n'a que deux choses à faire — exactement comme une rangée de dominos.

① Initialisation — on vérifie que P(0) est vraie (on pousse le 1ᵉʳ domino).
② Hérédité — on montre que si P(n) est vraie, alors P(n+1) l'est aussi (chaque domino fait tomber le suivant).
③ Conclusion — par récurrence, P(n) est vraie pour tout n (toute la rangée tombe).

La rédaction type

« Initialisation : pour n = 0, … donc P(0) est vraie. Hérédité : supposons P(n) vraie pour un certain n (hypothèse de récurrence). Montrons P(n+1)… Conclusion : d'après le principe de récurrence, P(n) est vraie pour tout entier n. »

Chapitre 6

Suites définies par récurrence : la toile d'araignée

Quand uₙ₊₁ = f(uₙ), on construit les termes « en escalier » entre la courbe y = f(x) et la droite y = x. Le point où elles se croisent est le point fixe ℓ : si la suite converge, c'est vers lui. Ici f est affine : f(x) = a·x + b.

point fixe ℓ = b/(1−a)

Tant que |a| < 1, l'escalier vient se coincer dans le coin : la suite converge vers ℓ. Si |a| > 1, l'escalier s'écarte et la suite diverge. C'est le même principe que les suites arithmético-géométriques, où l'on pose vₙ = uₙ − ℓ pour se ramener à une suite géométrique de raison a.

Exercices type Bac

Cherche d'abord au brouillon, puis déplie la correction.

★ ☆ ☆ — variation

1. Sens de variation

On considère la suite définie pour n ≥ 0 par uₙ = n² − 5n. Étudie son sens de variation.
Voir la correction

On calcule la différence uₙ₊₁ − uₙ :

uₙ₊₁ − uₙ = (n+1)² − 5(n+1) − (n² − 5n) = n² + 2n + 1 − 5n − 5 − n² + 5n = 2n − 4

Le signe de 2n − 4 : négatif pour n < 2, nul en n = 2, positif pour n > 2.

La suite est décroissante de u₀ à u₂, puis croissante à partir de n = 2. Elle n'est donc ni croissante ni décroissante sur ℕ.
★ ★ ☆ — récurrence

2. Démonstration par récurrence

Démontre que pour tout entier n ≥ 0 : 0 + 1 + 2 + … + n = n(n+1)/2.
Voir la correction

Soit P(n) : « 0 + 1 + … + n = n(n+1)/2 ».

Initialisation (n = 0) : le membre de gauche vaut 0, celui de droite 0×1/2 = 0. Donc P(0) est vraie.

Hérédité : supposons P(n) vraie pour un certain n. Alors :

0+1+…+n+(n+1) = n(n+1)/2 + (n+1) = (n+1)·(n/2 + 1) = (n+1)(n+2)/2

C'est bien la formule au rang n+1, donc P(n+1) est vraie.

Par récurrence, la propriété est vraie pour tout entier n ≥ 0.
★ ★ ☆ — suite géométrique

3. Reconnaître une géométrique

Soit (uₙ) définie par u₀ = 5 et uₙ₊₁ = 0,8 uₙ + 1. On pose vₙ = uₙ − 5.
  1. Montre que (vₙ) est géométrique ; précise sa raison et v₀.
  2. Exprime uₙ en fonction de n, puis donne la limite de (uₙ).
Voir la correction

Le point fixe vérifie ℓ = 0,8ℓ + 1, soit 0,2ℓ = 1, donc ℓ = 5 — d'où le choix vₙ = uₙ − 5.

1. vₙ₊₁ = uₙ₊₁ − 5 = 0,8uₙ + 1 − 5 = 0,8uₙ − 4 = 0,8(uₙ − 5) = 0,8 vₙ.

(vₙ) est géométrique de raison q = 0,8 et v₀ = u₀ − 5 = 0.

2. Comme v₀ = 0, on a vₙ = 0 pour tout n, donc uₙ = 5 (suite constante).

uₙ = 5 pour tout n, et lim uₙ = 5. (Si v₀ ≠ 0, on aurait uₙ = 5 + v₀·0,8ⁿ → 5, car 0 < 0,8 < 1.)
★ ★ ★ — limite

4. Limite par encadrement

Soit uₙ = (sin n)/n pour n ≥ 1. Détermine la limite de (uₙ).
Voir la correction

Pour tout n, −1 ≤ sin n ≤ 1. En divisant par n > 0 :

−1/n ≤ (sin n)/n ≤ 1/n

Or −1/n → 0 et 1/n → 0 quand n → +∞.

Par le théorème des gendarmes (encadrement), lim uₙ = 0.
Au Bac : pour une limite, cite toujours le théorème utilisé (gendarmes, comparaison, opérations, limite de qⁿ).
Avant de fermer le cahier

Les réflexes à retenir

  • Variation — étudie le signe de uₙ₊₁ − uₙ (ou compare uₙ₊₁/uₙ à 1 si termes > 0).
  • Arithmétique — uₙ = u₀ + n·r. Géométrique — uₙ = u₀·qⁿ.
  • Limite de qⁿ — → 0 si −1 < q < 1 ; → +∞ si q > 1 ; pas de limite si q ≤ −1.
  • Récurrence — initialisation, hérédité, conclusion. Toujours les trois.
  • Théorème des gendarmes — encadre uₙ entre deux suites de même limite.
  • uₙ₊₁ = a·uₙ + b — pose vₙ = uₙ − ℓ (point fixe) pour obtenir une suite géométrique.
