La fonction
logarithme

Le logarithme népérien ln transforme les produits en sommes — l'outil qui a permis de calculer avant les calculatrices. Réciproque de l'exponentielle, propriétés algébriques, courbe et limites, dérivée 1/x, croissances comparées, et la résolution d'équations. Chaque idée illustrée par une courbe à explorer.

Chapitre 1

ln, la réciproque de l'exponentielle

ln est la fonction qui « défait » exp : ln(eˣ) = x et e^(ln x) = x. Conséquence géométrique : leurs courbes sont symétriques par rapport à la droite y = x. Déplace le point et regarde son reflet.

y = ln x  ⇔  x = eʸ   (pour x > 0)   |   ln 1 = 0  ·  ln e = 1

Le point (t, eᵗ) sur l'exponentielle a pour reflet (eᵗ, t) sur le logarithme. C'est pourquoi ln n'existe que pour x > 0 (exp ne prend que des valeurs positives), et pourquoi ln est strictement croissante, comme exp.

Chapitre 2

La propriété magique : produit → somme

La règle fondatrice : ln(a × b) = ln a + ln b. Multiplier deux nombres revient à additionner leurs logarithmes. Toutes les autres formules en découlent. Règle a et b et regarde les longueurs s'additionner.

ln(ab) = ln a + ln b  ·  ln(a/b) = ln a − ln b  ·  ln(aⁿ) = n·ln a  ·  ln(√a) = ½·ln a

Chapitre 3

Courbe, variations et limites

ln est définie sur ]0 ; +∞[, strictement croissante, négative entre 0 et 1, positive au-delà. Aux bords : elle plonge vers −∞ près de 0 et monte (lentement) vers +∞.

limx→0⁺ ln x = −∞   (asymptote verticale x = 0)  |   limx→+∞ ln x = +∞

Repères utiles : ln 1 = 0 (la courbe coupe l'axe en x = 1) et ln e = 1 (avec e ≈ 2,718). Comme ln est strictement croissante, elle est bijective : ln a = ln b ⇔ a = b, et ln a < ln b ⇔ a < b. C'est ce qui autorise à « passer au ln » dans une équation.

Chapitre 4

Dérivée 1/x & croissances comparées

La dérivée de ln est très simple : (ln x)′ = 1/x. La pente de la tangente diminue quand x grandit — ln monte de plus en plus mollement. Tellement mollement que x finit toujours par l'emporter.

(ln x)′ = 1/x  ·  (ln u)′ = u′/u   |   limx→+∞ (ln x)/x = 0  ·  limx→0⁺ x·ln x = 0

Croissances comparées : à l'infini, toute puissance de x bat le logarithme — d'où (ln x)/x → 0. C'est l'outil qui lève les indéterminations « ∞/∞ » mêlant ln et polynômes, un grand classique du Bac.

Chapitre 5

Résoudre équations & inéquations

Deux réflexes : d'abord le domaine (tout ce qui est dans un ln doit être > 0), ensuite on utilise la bijection ou la fonction réciproque exp.

La méthode pas à pas

① Domaine : impose chaque argument > 0.

② Regrouper les ln d'un côté (avec les propriétés).

③ ln A = ln B ⇒ A = B  ·  ln A = k ⇒ A = eᵏ.

④ Vérifier que les solutions sont dans le domaine.

Les passages utiles

ln A = ln B  ⇔  A = B (avec A, B > 0)

ln A < ln B  ⇔  A < B (ln croissante)

ln X = k  ⇔  X = eᵏ

eˣ = k  ⇔  x = ln k (avec k > 0)

L'oubli du domaine est l'erreur classique : une « solution » qui rend un argument négatif est à rejeter.
À toi de jouer

Exercices type Bac

Cherche, puis déplie la correction.

★ ☆ ☆ — simplification

1. Simplifier une expression

Simplifie A = 2·ln 3 + ln 2 − ln 18.
Voir la correction

On utilise ln(aⁿ) = n·ln a puis produit/quotient :

A = ln(3²) + ln 2 − ln 18 = ln 9 + ln 2 − ln 18 = ln(9 × 2) − ln 18 = ln 18 − ln 18
A = 0.
★ ★ ☆ — équation

2. Résoudre une équation

Résous dans ℝ : ln(x) + ln(x − 3) = ln(10).
Voir la correction

Domaine : x > 0 et x − 3 > 0, donc x > 3.

On regroupe : ln(x(x−3)) = ln 10, puis par bijection x(x−3) = 10 :

x² − 3x − 10 = 0 → Δ = 9 + 40 = 49 x = (3 + 7)/2 = 5 ou x = (3 − 7)/2 = −2

−2 n'est pas dans le domaine (x > 3), on le rejette.

S = {5}.
★ ★ ☆ — inéquation

3. Résoudre une inéquation

Résous : ln(2x − 1) ≤ 1.
Voir la correction

Domaine : 2x − 1 > 0, soit x > 1/2.

On écrit 1 = ln e, puis on utilise la croissance de ln :

ln(2x − 1) ≤ ln e ⇔ 2x − 1 ≤ e ⇔ x ≤ (e + 1)/2

On croise avec le domaine x > 1/2. Or (e+1)/2 ≈ 1,859.

S = ] 1/2 ; (e + 1)/2 ].
★ ★ ★ — étude de fonction

4. Variations et limites

Soit f(x) = x − ln x sur ]0 ; +∞[.
  1. Calcule f′(x) et étudie son signe.
  2. Donne les limites de f en 0⁺ et en +∞, et dresse le tableau de variation.
Voir la correction

1. f′(x) = 1 − 1/x = (x − 1)/x. Sur ]0;+∞[, x > 0, donc le signe est celui de x − 1 : négatif sur ]0;1[, positif sur ]1;+∞[.

f décroît sur ]0 ; 1], croît sur [1 ; +∞[ ; minimum f(1) = 1 − 0 = 1.

2. En 0⁺ : x → 0 et −ln x → +∞, donc f(x) → +∞.

En +∞ : f(x) = x(1 − (ln x)/x). Or (ln x)/x → 0, donc 1 − (ln x)/x → 1, et x → +∞ :

lim0⁺ f = +∞ ; lim+∞ f = +∞ ; minimum global f(1) = 1.
Avant toute équation avec ln : écris le domaine. Après : vérifie que tes solutions y appartiennent.
Avant de fermer le cahier

Les réflexes à retenir

  • Définition — ln réciproque de exp : ln(eˣ)=x, e^(ln x)=x (x>0). ln 1=0, ln e=1.
  • Produit → somme — ln(ab)=ln a+ln b ; ln(a/b)=ln a−ln b ; ln(aⁿ)=n ln a.
  • Variations — strictement croissante sur ]0;+∞[ ; négative sur ]0;1[, positive après.
  • Limites — lim0⁺ ln x = −∞ (asymptote x=0) ; lim+∞ ln x = +∞.
  • Dérivée — (ln x)′ = 1/x ; (ln u)′ = u′/u.
  • Croissances comparées — (ln x)/x → 0 et x·ln x → 0 (x→0⁺).
  • Équations — domaine d'abord, puis ln A = ln B ⇔ A = B, ou ln X = k ⇔ X = eᵏ.
